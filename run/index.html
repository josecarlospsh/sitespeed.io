How to run

Standard (fetching one level of pages)

By default, sitespeed will crawl all the links that exist on the page, and fetch the pages that are on the same domain as the start. 

$ ./sitespeed.io http://yoursite.com

Or of course, if you want to test a specific page 

$ ./sitespeed.io http://yoursite.com/my/path

Fetching more than one level

Add another parameter

$ ./sitespeed.io http://yoursite.com 2

Fetching servers behind basic auth

Add login and password as next parameters. Note that you today always need to supply the crawl deep 

$ ./sitespeed.io http://yoursite.com 1 login password
