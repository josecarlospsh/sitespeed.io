How speedy is your site?

Analyze your site

Sitespeed is tool to analyze your site web performance and give you hints of what you should optimize. It collects data from multiple pages on your site (crawling from a startpoint), analyze it and output the result in a couple of HTML-files.

Sitespeed uses Yslow performance rules as a base to analyze a page. However, the rules has been tweaked and new rules has been added, check them out.

Run

1. Install PhantomJs

2. Install/use Java 1.7

3. Clone the project from Github 

git clone git@github.com:soulgalore/sitespeed.io.git

or download (v0.8) the zip.

4. Run the script 

$ ./sitespeed.io http://yoursite.com

Beta status

Sitespeed is beta until version 1.0, checkout issues and coming features at Github.

The current version has been verified on Mac OS X 10.8.1 & Ubuntu Linux 11.10.
