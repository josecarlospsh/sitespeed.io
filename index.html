How speedy is your site?

Analyze your site

Sitespeed is tool to analyze your sites wep performance and give you hints of what you should optimize. It collects data from multiple pages on your site (crawling from a startpoint), analyze it and output the result in a couple of HTML-files. I

Run

  • Install PhantomJs
  • Install/use Java 1.7
  • Clone the project from Github 
    git clone git@github.com:soulgalore/sitespeed.io.git
  • Run the script 
    $ ./sitespeed.io http://yoursite.com

Beta status

Sitespeed is early beta , checkout issues and coming features at Github.

The current version has been verified on Mac OS X 10.8 & Ubuntu Linux 11.10.
